Source context: AUMC Child Development Center — Two's Class, Summary of Developmental Progress, June 2026.

This document converts the classroom assessment into an app-friendly milestone structure. It is not a medical diagnosis. It is a practical tracking layer for parent planning, school follow-up, and age-banded developmental review.

# 2. How to Separate the Age Categories

The assessment comes from a Two's Class, but the app should not treat every skill as simply "age 2" or "age 3."

A better structure is:

## 2-Year Baseline

Skills expected or commonly emerging around age 2.

Use for:

- basic emotional regulation with adult help
- simple peer play
- running, moving, balancing
- using simple words or short phrases
- following simple directions
- showing interest in books
- early pretend play
- beginning counting interest

## 30-Month Bridge

Skills between toddler and preschool development.

Use for:

- more independent self-regulation
- more intentional peer interaction
- stronger expressive vocabulary
- two-step directions
- early problem solving
- symbolic play
- early shape / number recognition
- longer attention span

## 3-Year Stretch

Skills to encourage as the child approaches preschool maturity.

Use for:

- sharing and turn-taking with less support
- using 3- to 4-word sentences
- retelling simple events
- asking and answering questions about books
- identifying basic shapes
- counting with more accuracy
- using drawing/writing tools with more control
- beginning to solve peer problems verbally

# 3. Status Labels for the App

Use these instead of only "passed" or "failed."

```text
Accomplished
Emerging
Needs Practice
Not Yet Observed
Parent / Teacher Goal
```

Recommended meaning:

## Accomplished

The skill is consistently demonstrated or marked as mastered.

## Emerging

The skill is partially present or appears with support.

## Needs Practice

The skill is not yet consistent and should become a near-term goal.

## Not Yet Observed

The skill was not marked or needs more observation.

## Parent / Teacher Goal

The teacher specifically identified this as a growth target.

# 11. Recommended Parent Priority Map

## Highest Priority: Language Expansion

Reason:

```text
Robbie is using expressions, gestures, and one-word phrases. The next developmental step is expanding spoken vocabulary, phrase length, and conversational turns.
```

Suggested daily activities:

```text
Narrate routines
Offer choices using words
Expand his phrases
Read short books aloud
Ask simple either/or questions
Repeat and extend his words
Sing rhyming songs
Name objects during play
```

Example:

```text
Robbie says: "ball"
Parent says: "big ball" or "roll ball" or "red ball"
```

## Second Priority: Social Problem Solving

Reason:

```text
Robbie seeks adult help and responds to support. The next step is helping him use words and simple scripts during peer conflict.
```

Suggested scripts:

```text
My turn.
Help please.
Stop.
More please.
Can I play?
I want ___.
You can have it next.
```

## Third Priority: Early Literacy

Reason:

```text
Robbie enjoys books, songs, rhymes, and storytime. This is a strength that can support language growth.
```

Suggested activities:

```text
Read daily
Point to pictures
Ask "Where is ___?"
Let him turn pages
Pause during rhymes
Practice name letters casually
Let him scribble after reading
```

## Fourth Priority: Early Math / Shapes / Numbers

Reason:

```text
Robbie is interested in counting, but number recognition, quantities, shapes, and patterns are still developing.
```

Suggested activities:

```text
Count stairs
Count snacks
Sort toys by color
Match shapes
Point out circles and squares
Use first / next / last language
Copy simple patterns
```

# 16. Developmental Reference Layer

Use CDC milestone checklists as a reference layer, not as the only source of truth. CDC milestones are designed to help monitor how children play, learn, speak, act, and move, and the CDC provides age checklists from infancy through age 5.

Recommended app reference anchors:

```text
2 years
30 months
3 years
```

The app should allow each milestone to be connected to:

```text
teacher observation
parent observation
CDC age anchor
pediatrician note, if needed
follow-up activity
```

# 17. Final Principle

This assessment should not be used to label Robbie.

It should be used to guide formation.

The app should show:

```text
What is already strong.
What is emerging.
What needs practice.
What adults should support next.
```

The goal is not pressure.

The goal is better attention.
